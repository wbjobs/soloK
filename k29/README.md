# 声波测井数据可视化解释平台

一个功能完整的声波测井数据处理、解释和可视化平台，支持多井对比、时深转换、弹性参数计算、孔隙度预测、岩性判别、裂缝识别等专业功能。

## ✨ 功能特性

### 📁 数据加载
- 支持标准LAS文件格式加载
- 批量多井数据管理
- 内置示例数据快速体验
- 支持数据预览和统计分析

### ⚙️ 声波数据处理
**地层弹性参数计算**
- 纵波速度(VP)、横波速度(VS)
- 泊松比、杨氏模量、体积模量、剪切模量
- 拉梅常数、声波阻抗

**孔隙度预测**
- Wyllie时间平均方程
- Raymer-Hunt-Gardner公式
- 密度孔隙度
- 综合孔隙度

**脆性指数计算**
- 基于杨氏模量和泊松比归一化
- 适用于页岩气评价
- 脆性等级自动分类

**岩性自动判别**
- 随机森林机器学习算法
- 支持6种岩性：砂岩、泥岩、石灰岩、白云岩、煤、盐岩
- 置信度评估

**裂缝识别**
- 基于横波分裂现象
- 快慢横波时差分析
- 裂缝强度和等级划分

### 📈 可视化
**综合测井曲线图**
- 多道曲线自定义显示
- 深度范围选择
- 交互式缩放、平移
- 悬停数据显示

**纵横波交会图**
- VP-VS、模量、泊松比等参数交会
- 彩色映射密度、伽马、岩性
- 支持任意参数组合

**3D井轨迹**
- 三维井眼轨迹可视化
- 井轨迹俯视图
- 支持多井对比显示
- 可配置方位角、倾角、造斜率

**多井对比**
- 同井场2-4口井平行对比
- 任意曲线选择对比
- 数据统计对比表

### ⏱️ 时深转换
- 深度域到时间域转换
- Checkshot数据标定
- 时间域测井曲线显示
- 平均速度曲线

### 📤 数据导出
- Excel多Sheet导出
- PDF测井解释报告
- 解释成果表自动生成

## 🚀 快速开始

### 环境要求
- Python 3.8+
- Windows / Linux / macOS

### 安装依赖
```bash
pip install -r requirements.txt
```

### 启动应用
```bash
streamlit run app.py
```

应用将在浏览器中自动打开，默认地址: http://localhost:8501

## 📖 使用指南

### 1. 加载数据
- **方式1**: 点击左侧边栏"浏览文件"上传LAS文件
- **方式2**: 点击"加载示例数据(3口井)"快速体验

### 2. 数据处理
在"⚙️ 数据处理"页面依次执行：
1. 弹性参数计算
2. 孔隙度预测（可调整骨架参数）
3. 脆性指数计算
4. 岩性自动判别
5. 裂缝识别

### 3. 可视化分析
在"📈 可视化"页面查看：
- 综合测井曲线图（自定义道数和曲线）
- 纵横波交会图（分析岩石物理特征）
- 3D井轨迹（配置井眼轨迹参数）
- 多井对比（选择对比曲线）

### 4. 时深转换
在"⏱️ 时深转换"页面：
- 执行深度域到时间域转换
- 输入Checkshot数据进行标定
- 查看时间域测井曲线

### 5. 导出成果
在"📤 数据导出"页面：
- 选择需要导出的Sheet
- 下载Excel文件
- 生成并下载PDF解释报告

## 🔧 技术栈

- **UI框架**: Streamlit
- **数据处理**: NumPy, Pandas
- **数据可视化**: Plotly
- **机器学习**: Scikit-learn
- **文件处理**: lasio, openpyxl, ReportLab

## 📝 注意事项

1. LAS文件需包含必要的曲线：
   - 弹性参数计算: DT, DTS, RHOB
   - 孔隙度预测: DT, RHOB
   - 岩性判别: DT, RHOB, GR
   - 裂缝识别: DTS_FAST, DTS_SLOW (或DTS)

2. 示例数据为合成数据，仅用于功能演示

3. 实际应用中请根据工区情况调整模型参数
